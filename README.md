# ISTA-RegFox
Управление режимом программирования BMW ISTA через реестр Windows — с автобэкапами, откатом и защитой от дублей. Самодостаточный .exe, только встроенные библиотеки Python. |   Registry-based BMW ISTA programming mode controller — with auto-backups, rollback, and deduplication. Self-contained .exe, built with standard Python libraries only.
